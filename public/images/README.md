# About Page Images

Add the following images to this folder for the About page:

## Required Images:

1. **about-hero.jpg** (1920x1080px or larger)
   - Wide shot of cannabis field or grow operation
   - Will be used as hero background
   - Should be high quality and atmospheric

2. **cannabis-grow.jpg** (800x600px or larger)
   - Cannabis plants growing in controlled environment
   - Shows quality and care in cultivation
   - Used in "Our Mission" section

3. **quality-control.jpg** (800x800px or larger)
   - Close-up of premium cannabis buds or quality testing
   - Professional looking
   - Used in "Quality Commitment" section

## Placeholder Images:

Until you add your own images, you can use placeholder images from:
- Unsplash: https://unsplash.com/s/photos/cannabis
- Pexels: https://www.pexels.com/search/cannabis/

Or create placeholder images with the same dimensions.

## Image Optimization:

For best performance:
- Use WebP format when possible
- Keep file sizes under 500KB
- Optimize with tools like TinyPNG or ImageOptim
